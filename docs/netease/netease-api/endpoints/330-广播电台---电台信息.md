# 广播电台 - 电台信息

> **接口路径（摘自原文）:** `/broadcast/channel/currentinfo`

**文档说明:** 接口与 [NeteaseCloudMusicApi](https://gitlab.com/Binaryify/NeteaseCloudMusicApi) 约定一致；正文来自仓库内 `docs/raw/home.md` 的离线副本并按章节拆分。API 基地址由应用配置提供，本仓库不包含任何服务端点。

---


说明: 调用此接口, 传入电台id, 获取广播电台 - 电台信息

**必选参数：**

`id`: 电台id

**接口地址:** `/broadcast/channel/currentinfo`

**调用例子:** `/broadcast/channel/currentinfo?id=5`
