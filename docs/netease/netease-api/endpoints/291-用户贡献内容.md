# 用户贡献内容

> **接口路径（摘自原文）:** `/ugc/detail`

**文档说明:** 接口与 [NeteaseCloudMusicApi](https://gitlab.com/Binaryify/NeteaseCloudMusicApi) 约定一致；正文来自仓库内 `docs/raw/home.md` 的离线副本并按章节拆分。API 基地址由应用配置提供，本仓库不包含任何服务端点。

---

说明: 登录后调用此接口,使用此接口,可获取当前登录用户贡献内容

**必选参数：**  

`type`: 内容种类
分为以下几种类型:
曲库纠错 歌手:1 专辑:2 歌曲:3 MV:4 歌词:5 翻译:6
曲库补充 专辑:101 MV:103

**可选参数：** 
`limit`: 取出条目数量 , 默认为 10

`offset`: 偏移数量

`auditStatus`: 审核状态
待审核:0 未采纳:-5 审核中:1 部分审核通过:4 审核通过:5

`order`: 排序,默认为降序 降序:desc 顺序:asc

**接口地址:** `/ugc/detail`

**调用例子:** `/ugc/detail`
