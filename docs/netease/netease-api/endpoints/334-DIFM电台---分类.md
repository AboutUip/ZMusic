# DIFM电台 - 分类

> **接口路径（摘自原文）:** `/dj/difm/all/style/channel`

**文档说明:** 接口与 [NeteaseCloudMusicApi](https://gitlab.com/Binaryify/NeteaseCloudMusicApi) 约定一致；正文来自仓库内 `docs/raw/home.md` 的离线副本并按章节拆分。API 基地址由应用配置提供，本仓库不包含任何服务端点。

---


说明: 调用此接口, 获取DIFM电台分类

**必选参数 :**  

`sources`: 来源列表, 0: 最嗨电音 1: 古典电台 2: 爵士电台

**接口地址:** `/dj/difm/all/style/channel`

**调用例子:** `/dj/difm/all/style/channel?sources=[0]`
