# 广播电台 - 全部电台

> **接口路径（摘自原文）:** `/broadcast/channel/list`

**文档说明:** 接口与 [NeteaseCloudMusicApi](https://gitlab.com/Binaryify/NeteaseCloudMusicApi) 约定一致；正文来自仓库内 `docs/raw/home.md` 的离线副本并按章节拆分。API 基地址由应用配置提供，本仓库不包含任何服务端点。

---


说明: 调用此接口, 获取广播电台 - 全部电台

**可选参数 :**

`categoryId` : 类别id, 默认为 0，可从“广播电台 - 分类/地区信息”接口获取

`regionId` : 地区id, 默认为 0，可从“广播电台 - 分类/地区信息”接口获取

**接口地址:** `/broadcast/channel/list`

**调用例子:** `/broadcast/channel/list`
